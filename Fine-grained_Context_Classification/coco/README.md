# COCO Dataset

Download COCO 2017 Train images and annotations from https://cocodataset.org/#download and organize as follows:

```
coco/
├── images/
│   └── train2017/       # 118K training images
└── annotations/
    └── instances_train2017.json
```

Required files:
- **Train images**: `train2017.zip` (18GB)
- **Annotations**: `annotations_trainval2017.zip` (241MB)
