# Location LoRA Training

Fine-tunes Qwen2.5-VL-3B-Instruct with LoRA to classify whether an object is in-context or out-of-context based on its **location** (physically and contextually reasonable placement).

## Usage

```bash
bash launch_training.sh
```

Requires the cached dataset at `../cached_dataset/` (generated by `../prepare_dataset.ipynb`).

## Pre-trained LoRA Adapter

Download the trained adapter from HuggingFace and place in `final_model/`:

Source: https://huggingface.co/COinCO/Context_Classification_Models/tree/main/location_lora

```bash
huggingface-cli download COinCO/Context_Classification_Models --include "location_lora/*" --local-dir .
mv location_lora qwen25_finetuned_location_specialist_multi_gpu/final_model
```
