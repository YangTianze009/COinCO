# Co-occurrence LoRA Training

Fine-tunes Qwen2.5-VL-3B-Instruct with LoRA to classify whether an object is in-context or out-of-context based on **co-occurrence** with other objects in the scene.

## Usage

```bash
bash launch_training.sh
```

Requires the cached dataset at `../cached_dataset/` (generated by `../prepare_dataset.ipynb`).

## Pre-trained LoRA Adapter

Download the trained adapter from HuggingFace and place in `final_model/`:

Source: https://huggingface.co/COinCO/Context_Classification_Models/tree/main/co_occurrence_lora

```bash
huggingface-cli download COinCO/Context_Classification_Models --include "co_occurrence_lora/*" --local-dir .
mv co_occurrence_lora qwen25_finetuned_co_occurrence_specialist_multi_gpu/final_model
```
